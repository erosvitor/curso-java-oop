## Sobre
O projeto é um curso que ensina os fundamentos da programação orientada a objetos na linguagem Java.

## Conteúdo do curso
* Classes
* Atributos
* Métodos
* Construtores
* Sobrecarga de construtores
* Sobrecarga de métodos
* Encapsulamento
* Métodos estáticos
* Herança
* Lista de objetos
