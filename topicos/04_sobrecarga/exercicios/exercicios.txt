Exercício 01 - Tabela de Preços
--------------------------------------------------------------------------------
Sobrecarregar cada um dos métodos da classe Produto de tal forma que seja possível
passar a taxa a ser utilizada na definição de cada preço. Obs. Os métodos atuais 
devem ser mantidos.


Exercício 02 - Calendario
--------------------------------------------------------------------------------
Criar uma classe para manipular datas.

Instruções:

- Criar um projeto chamado Calendario
- Criar uma classe chamada Calendario contendo os seguintes métodos:
  - Um método que receba o mês como parâmetro e retorne o nome do mês por extenso
  - Um método que receba uma data no formato string e retorne o nome do mês por
    extenso.
